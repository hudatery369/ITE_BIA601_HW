**الطلاب:**
- أسامة الحوري
- روان طلال الفيحان
- هدى محمد طيري
- أحمد علاوي
- حسن
- طارق نصر 
- يزن
- أحمد

---

## التثبيت والتشغيل

### 1. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 2. تشغيل التطبيق
```bash
python app.py
```

### 3. فتح المتصفح
افتح: `http://localhost:5000`

---

## الاستخدام

1. حمّل ملف البيانات (CSV أو Excel)
2. اختر عمود الهدف (Target)
3. اضبط معاملات الخوارزمية الجينية
4. اضغط "تشغيل الخوارزمية الجينية"
5. شاهد النتائج
